WebLabeler 是一款便捷的工具，适用于每天需要在多个环境（development, testing, staging, and production）之间切换的 Web 开发人员、测试人员、内容管理员等。

借助这个轻量级扩展，您可以轻松创建醒目的标识，使您的标签页易于区分， 帮助您避免混淆和代价高昂的错误。

使用 WebLabeler 提高生产力并保持井然有序！💪✨

主要功能：

🎨 完全可自定义的标签：通过调整形状（三角形、丝带或 边框）、颜色、位置和不透明度，精确设计您想要的标签。

🔖 网站图标徽章：直接在浏览器标签栏中即时区分项目。

🌐 强大的域名规则：使用灵活的规则类型，根据项目的域名自动应用标签。

🖱 通过 Chrome 弹出窗口快速访问：轻松开关标签，无需导航到设置。

📦 导出/导入配置：与您的团队共享标签设置或在项目之间重复使用。支持 URL 同步 和自动更新。

📥 轻松迁移：从其他扩展迁移数据，立即开始使用 WebLabeler。

🔄 跨设备同步：使用 Google Chrome 的账户同步功能，在所有设备上保持标签配置 同步。

🌙 深色模式支持：享受深色模式选项，在深夜或低光环境中 舒适使用。
