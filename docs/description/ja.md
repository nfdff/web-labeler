WebLabelerは、日々複数の環境（development, testing, staging, and production）を扱う ウェブ開発者、テスター、コンテンツマネージャーなどのための便利なツールです。

この軽量な拡張機能を使用すると、タブを簡単に区別できる目立つインジケーターを簡単に 作成でき、混乱や高コストなミスを避けるのに役立ちます。

WebLabelerで生産性を向上させ、整理整頓しましょう！💪✨

主な機能：

🎨 完全にカスタマイズ可能なラベル：形状（三角形、リボン、または フレーム）、色、位置、不透明度を調整して、ラベルを思い通りにデザインできます。

🔖 ファビコンバッジ：ブラウザのタブバーで直接プロジェクトを即座に区別できます。

🌐 強力なドメインルール：柔軟なルールタイプを使用して、ドメイン名に基づいてプロジェクトにラベルを自動的に適用 します。

🖱 Chromeポップアップからのクイックアクセス：設定に移動せずにラベルを簡単にオン・オフできます。

📦 設定のエクスポート/インポート：チームとラベル設定を共有したり、プロジェクト間で再利用したりできます。自動更新付きのURL同期 をサポートします。

📥 簡単な移行：他の拡張機能からデータを移行して、すぐにWebLabelerの使用を開始できます。

🔄 デバイス間で同期：Google Chromeのアカウント同期機能を使用して、すべてのデバイスでラベル設定を同期 させます。

🌙 ダークモードのサポート：夜間のセッションや低照度環境で快適に使用できるダークモードオプション を楽しめます。
