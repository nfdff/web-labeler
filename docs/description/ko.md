WebLabeler는 매일 여러 환경 (development, testing, staging, and production)을 다루는 웹 개발자, 테스터, 콘텐츠 관리자 등을 위한 편리한 도구입니다.

이 가벼운 확장 프로그램을 사용하면 탭을 쉽게 구별할 수 있는 눈에 띄는 표시를 손쉽게 만들어 혼란과 비용이 많이 드는 실수를 피할 수 있습니다.

WebLabeler로 생산성을 높이고 체계적으로 유지하세요! 💪✨

주요 기능:

🎨 완전히 사용자 정의 가능한 레이블: 모양(삼각형, 리본 또는 프레임), 색상, 위치 및 불투명도를 조정하여 원하는 대로 레이블을 디자인할 수 있습니다.

🔖 파비콘 배지: 브라우저 탭 표시줄에서 직접 프로젝트를 즉시 구분할 수 있습니다.

🌐 강력한 도메인 규칙: 유연한 규칙 유형을 사용하여 도메인 이름을 기반으로 프로젝트에 레이블을 자동으로 적용 합니다.

🖱 Chrome 팝업을 통한 빠른 액세스: 설정으로 이동하지 않고도 레이블을 쉽게 켜거나 끌 수 있습니다.

📦 구성 내보내기/가져오기: 팀과 레이블 설정을 공유하거나 프로젝트 간에 재사용할 수 있습니다. 자동 업데이트가 포함된 URL 동기화를 지원합니다.

📥 쉬운 마이그레이션: 다른 확장 프로그램에서 데이터를 마이그레이션하고 즉시 WebLabeler 사용을 시작할 수 있습니다.

🔄 기기 간 동기화: Google Chrome의 계정 동기화 기능을 사용하여 모든 기기에서 레이블 구성을 동기화된 상태로 유지합니다.

🌙 다크 모드 지원: 심야 세션이나 저조도 환경에서 편안하게 사용할 수 있는 다크 모드 옵션을 즐기세요.
